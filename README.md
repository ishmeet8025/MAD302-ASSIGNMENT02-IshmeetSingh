Project Description

This Android application displays a list of products using RecyclerView. Users can tap on any product to navigate to a detail screen where full product information is displayed. The app demonstrates interaction handling, navigation using Intent, and data passing between activities.

---

Features

* Displays at least 6 products (name, price, description)
* Uses RecyclerView with Adapter and ViewHolder
* Implements lambda-based click handling
* Navigates between screens using Intent
* Passes product data between activities
* Applies null safety when receiving data

---

Technologies Used

* Kotlin
* Android Studio
* RecyclerView
* Intents

---

Project Structure

* MainActivity.kt – Displays product list
* Product.kt – Data model class
* ProductAdapter.kt – RecyclerView adapter
* DetailActivity.kt – Displays selected product details
* activity_main.xml – Main UI layout
* activity_detail.xml – Detail screen layout
* item_product.xml – Individual product item layout
